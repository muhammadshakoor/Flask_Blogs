---

## ✨ Features

- 🐍 Flask application structure with `app.py` and `main.py`
- 🎨 HTML/CSS/JS frontend with Jinja2 templating
- 📄 Multiple route handling and page rendering
- 🗂️ Organised with `static/` and `templates/` directories
- ⚙️ Virtual environment setup included

---

## 📁 Project Structure

```
Flask_Blogs/
├── templates/      # Jinja2 HTML templates
├── static/         # CSS, JavaScript, images
├── env/            # Python virtual environment
├── app.py          # Main Flask application
├── main.py         # Entry point / additional routes
└── .vscode/        # VS Code settings
```

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/muhammadshakoor/Flask_Blogs.git
cd Flask_Blogs

# Create and activate virtual environment
python -m venv env
source env/bin/activate       # Linux/Mac
env\Scripts\activate          # Windows

# Install Flask
pip install flask

# Run the app
python app.py
```

App runs at `http://127.0.0.1:5000`
